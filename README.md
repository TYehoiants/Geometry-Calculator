# Geometry-Calculator
A simplest desktop application built with C# and WinForms for calculating the area and perimeter of various geometric shapes.




## Features
* [cite_start]**Multiple Shapes:** Supports calculations for circles, squares, rectangles, and triangles[cite: 8].
* [cite_start]**Real-time Results:** Calculates both Area (Fläche) and Perimeter (Umfang) instantly[cite: 8].
* [cite_start]**Input Validation:** Ensures only valid numerical data is processed to prevent application crashes[cite: 8].


##  Tech Stack
* [cite_start]**Language:** C# 
* [cite_start]**Framework:** .NET / WinForms 
* [cite_start]**IDE:** Visual Studio [cite: 8]

## 📖 How it Works
The application uses Object-Oriented Programming (OOP) principles. Each geometric shape is handled by specific logic that takes user-defined parameters (like radius or side length) and applies the corresponding mathematical formulas
