# Geometry-Calculator
A simplest desktop application built with C# and WinForms for calculating the area and perimeter of various geometric shapes.




## Features
**Multiple Shapes:** Supports calculations for circles, squares, rectangles, and triangles.
**Real-time Results:** Calculates both Area (Fläche) and Perimeter (Umfang) instantly.
**Input Validation:** Ensures only valid numerical data is processed to prevent application crashes


##  Tech Stack
**Language:** C# 
**Framework:** .NET / WinForms 
**IDE:** Visual Studio

## 📖 How it Works
The application uses Object-Oriented Programming (OOP) principles. Each geometric shape is handled by specific logic that takes user-defined parameters (like radius or side length) and applies the corresponding mathematical formulas
